# 概要
WPFを用いて作成した電卓プログラムです。  
  
![Image 1](https://github.com/Ryo-Takahashi-0422/Calculator/blob/main/examples/UI.png)
   
# 要インストール
- .NET 8.0以上
  
# 実行
ダウンロード後、...Calculator\bin\Release\net8.0-windows\Calculator.exeをダブルクリックして下さい。  
    
# 外部ライブラリ
- Caliburn Micro(ver 4.0.212)
